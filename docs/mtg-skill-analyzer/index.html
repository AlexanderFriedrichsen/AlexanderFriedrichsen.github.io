Deck Recommender

Adjust expected meta shares to see which deck has the best expected win rate against the field.

Methodology

This tool shows how often each MTG archetype beats every other archetype, then adjusts for player skill using ELO ratings built from match histories. If a deck's "raw" win rate is inflated because elite players pilot it, the "adjusted" view corrects for that. The "delta" view reveals which decks benefit most from having skilled pilots, and which decks' power is inherent regardless of who plays them.
Data sources and processing details

Data Sources

Tournament data from 944 events across melee.gg, MTGO, and spicerack.gg (Dec 2025 – Mar 2026). 68K+ matches across 6 formats. Archetype classification via signature card matching and source labels. Base player ratings from MTG ELO Project (1,099 rated players).

ELO Rating Pipeline

Comprehensive ELO system built from 60K+ tabletop match results:

  • 1,099 players initialized with their scraped ELO from mtgeloproject.net
  • All other players start at 1500 ELO
  • All tabletop tournament matches processed chronologically with K=36 (MTGO excluded from ELO — only has playoff data)
  • ELO divisor = 1135.77 (200 points = 60% win expectancy, matching mtgeloproject.net)
  • Result: 12,528 players (95.2%) have non-default ELO ratings
  • ELO coverage per match: 57–94% depending on format

Skill Adjustment (Residual Method)

  • For each match, compute expected win probability from ELO: E = 1 / (1 + 10^((R_opp - R_player) / 1135.77))
  • Compute residual: actual result (1 or 0) minus expected probability
  • Skill-adjusted win rate = 50% + average residual for that matchup cell
  • This removes the bias where strong players inflate their deck's apparent win rate

Confidence Intervals

Wilson score intervals at 95% confidence are shown in tooltips. These are more reliable than simple percentage intervals for small sample sizes.

Skill Sensitivity

Average absolute delta (adjusted - raw) across all matchups for each archetype. High sensitivity means the deck performs very differently depending on pilot skill.

Limitations

  • MTGO tournaments lack archetype classifications, so only tabletop (melee.gg) data has full matchup info for most formats
  • Standard has the most complete data with both MTGO and tabletop coverage
  • Vintage and Premodern data is sparse; results may be unreliable
  • ELO ratings are more reliable for players with many matches; newer players may have inaccurate ratings
  • Mirror matches are excluded from the matrix
